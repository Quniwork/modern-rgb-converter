# CSS Modern Color Converter

🚀 一個簡單強大的 VS Code 外掛，專門將舊版的 `rgba(r, g, b, a)` 顏色格式轉換為現代 CSS 語法 `rgb(r g b / a%)`。

## ✨ 功能特色

* **圈選轉換**：只針對你選取的顏色代碼進行格式化。
* **全文件掃描**：一鍵轉換整個 CSS/JS 檔案中的所有舊版顏色格式。
* **自動百分比計算**：自動將透明度（例如 `0.4`）轉換為百分比（例如 `40%`）。


## 📦 安裝教學 (Installation)

你可以透過以下兩種方式安裝此擴充套件：

### 1. 透過 VSIX 檔案安裝 (推薦)
如果你已經下載了 `.vsix` 安裝包，可以按照以下步驟操作：
1. 開啟 **VS Code**。
2. 進入側邊欄的 **「擴充功能 (Extensions)」** 視圖 (快捷鍵：`Ctrl+Shift+X`)。
3. 點擊右上角的 **「...」** (更多動作) 圖示。
4. 選擇 **「從 VSIX 安裝... (Install from VSIX...)」**。
5. 選取你封裝好的 `css-color-modernizer-x.x.x.vsix` 檔案即可完成。

### 2. 在 Antigravity 或開發環境中手動載入
如果你是在 **Antigravity** 或本地開發環境中使用：
1. 將整個專案資料夾放入你的工作區。
2. 確保資料夾內包含 `package.json` 與 `extension.js`。
3. 在該資料夾開啟終端機，執行 `npm install` (若有依賴套件)。
4. 按下 **`F5`** 啟動「VS Code 延伸模組開發」視窗進行即時測試與使用。

## 🔍 診斷與調試 (Debug Tool)

根據開發規範，本工具內建了診斷函式：
- 若執行轉換時沒有反應，請開啟 **「切換開發人員工具 (Toggle Developer Tools)」** 查看 Console 輸出。
- 工具會自動檢查「欄位匹配」邏輯，若無法識別顏色格式，會彈出視窗提示具體缺漏，避免無效操作。

## 🛠 使用方法

### 1. 右鍵選單
- **圈選範圍**：選取一段包含 `rgba` 的文字，點擊右鍵選擇 `轉換顏色：僅圈選範圍 (Modern RGB)`。
- **全文件**：在編輯器任何地方點擊右鍵，選擇 `轉換顏色：掃描並取代全文件 (Modern RGB)`。

### 2. 快捷鍵
- **僅圈選範圍**：
    - Mac: `Cmd + Shift + M`
    - Windows: `Ctrl + Shift + M`
- **掃描全文件**：
    - Mac: `Cmd + Alt + M`
    - Windows: `Ctrl + Alt + M`

## 📝 轉換範例

| 原始格式 | 轉換後格式 |
| :--- | :--- |
| `rgba(255, 0, 0, 0.5)` | `rgb(255 0 0 / 50%)` |
| `rgb(0, 0, 0)` | `rgb(0 0 0)` |
| `rgba(10, 20, 30, 1)` | `rgb(10 20 30 / 100%)` |
